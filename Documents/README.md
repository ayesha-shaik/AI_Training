# Documents

Prepared study materials — notes, summaries, cheat-sheets, practice sets — generated during learning sessions.

## Naming convention

```
YYYY-MM-DD_<SessionName>_<DocType>.<ext>
```

- **YYYY-MM-DD** — date the doc was prepared (e.g. `2026-06-27`)
- **SessionName** — the topic/session in PascalCase (e.g. `LinearRegression`, `KMeans`, `IQR-Outliers`)
- **DocType** — what kind of doc: `Notes`, `Summary`, `CheatSheet`, `Practice`, `Quiz`, `Flashcards`
- **ext** — `.md` for notes, `.html` for interactive, `.pdf` if exported

### Examples
```
2026-06-27_LinearRegression_Notes.md
2026-06-27_IQR-Outliers_CheatSheet.md
2026-07-04_SVM_Practice.md
2026-07-18_KMeans_Summary.md
```

This keeps docs sorted chronologically and instantly searchable by topic.
